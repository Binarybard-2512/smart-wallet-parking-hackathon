# Smart Wallet Parking

A Smart Parking System designed and developed for a hackathon.

## Features
- **Frontend:** Modern, dynamic web interface.
- **Backend:** Python-based service.
- **IoT Capabilities:** Integrates seamlessly with ESP32-based hardware for intelligent sensing and automated parking validation.

## Setup Instructions

### Frontend
1. Navigate to the `frontend` directory: `cd frontend`
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

### Backend
1. Navigate to the `backend` directory: `cd backend`
2. Set up a virtual environment: `python -m venv venv`
3. Activate the virtual environment.
4. Install requirements and start the backend service.
